# Life Agent 🚀

一个基于多 Agent 架构的 AI 行为分析与成长系统。

## ✨ 功能

- 自动记录用户行为
- 多轮历史分析（长上下文）
- AI生成个性化计划
- 行动驱动反馈机制

## 🧠 架构

- Input Agent：数据采集
- Analysis Agent：行为分析
- Planning Agent：策略生成
- Action Agent：执行驱动

## 🚀 启动

```bash
pip install -r requirements.txt
python main.py
```

## 🔑 环境变量

API_KEY=你的key
BASE_URL=小米API地址
